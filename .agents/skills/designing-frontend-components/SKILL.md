# Designing Frontend Components

This skill focuses on building high-quality frontend components for the VeXeViet platform.

## Capabilities
- Shadcn/ui integration
- Tailwind CSS styling
- Responsive & Accessible design (WCAG 2.1 AA)
- Storybook component documentation

## Standards
- Use named exports
- Absolute imports (`@/components/...`)
- Strict TypeScript types
- 80%+ test coverage for components
